# 原生JS：移动端音乐播放器

## 1. 技术栈

* 移动端适配 —— 上、下部分设置固定高度（px），中间部分用“屏幕的高度（视口单位 vh）”减去“上、下高度之和”。
* 用 Flex 布局
* 用 SVG 设置图标，可以是多色
* 用 ES6 语法
* 用 SASS
* 用“打包工具”—— Parcel

***

## 2. 实现功能

* 左右滑动 —— 两个面板
* 加载“歌曲信息” —— 歌曲名、歌手、专辑
* 加载“歌词”
* 播放/暂停
* 上一曲
* 下一曲
* 刷新当前歌曲
* 定位“歌词” —— 跟随播放滚动，高亮歌词
* 点击进度条后进行相应播放
* 拖动进度条按钮后进行相应播放
